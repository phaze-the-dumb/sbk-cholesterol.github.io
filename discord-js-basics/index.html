discord.js basics

Getting things ready

What you need:

- NodeJS Note: The latest version is V14.13.1, but you are recommended to download the V12.19.0 one.

- A text editor e.g. Visual Studio Code, Atom. Do not use Notepad to edit the file!

- 64-bit system.

Get ready to code

You are strongly recommended to learn JavaScript basics. It would be confusing for you if you know nothing about JavaScript.

1. Create an empty folder in your computer

2. Start your code editor

3. Open the empty folder you created before.

4. Create a [YOUR_FILE_NAME].js file MUST BE A .js FILE!

5. Start a terminal, type npm install discord.js and hit enter

6. Type npm init -y and hit enter
